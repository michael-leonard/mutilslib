Generates a list of files with a given extension in a given path uses command prompt + dos syntax writes dos output to a temp file and then deletes requisite dos functions: dir Author: Originally Written by Michael Leonard as Generate_RscriptList Generalised to Generate_FileList by Mark Thyer, May 2009.

Parameters:
path the path to the files are located
ext the extension of the files that need to be returned, for files with one letter extentions (e.g, *.r), it is important to use a space, e.g. ext="*.r " to distinguish from .r** extensions
listtype the type of list needed, options include "FILE" - list of files "DIRECTORY" - list of directories
